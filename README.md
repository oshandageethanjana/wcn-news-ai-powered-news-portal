# WCN News — AI-Powered Global News Portal

**WCN News** is an AI-powered global news portal developed by **WhiteCoder**. This project is built using **PHP, MySQL, HTML, CSS, JavaScript, and PHPMailer**. It provides a complete news publishing platform with user accounts, category-based news browsing, search, newsletter support, comments, reactions, recommendations, SEO pages, and automated news processing features.

## Live Demo

**Website:** https://news.whitecoder.online

## Project Overview

WCN News is designed as a modern online news platform where users can browse the latest global news, read articles, search news content, interact with posts, and subscribe to newsletters. The system also includes admin-side support for managing news content and related website features.

This project is suitable for portfolio showcase, academic project submission, and PHP/MySQL web application demonstration.

## Main Features

- User registration and login system
- Dynamic news article display
- Category-based news browsing
- News search functionality
- Single news article pages
- User profile support
- Reading history support
- Comments and reactions
- Newsletter subscription
- Email support using PHPMailer
- Recommendation system
- SEO-friendly sitemap and robots.txt
- Cron job support for automation
- Responsive front-end design
- PHP and MySQL database integration

## Installation Guide

After receiving the ZIP password:

1. Download the ZIP file from this repository.
2. Extract the ZIP file using the provided password.
3. Copy the extracted project folder to your server directory.
   - For XAMPP: `htdocs/`
   - For cPanel: `public_html/`
4. Create a MySQL database.
5. Import the database file if included.
6. Update the database connection details in the configuration file.
7. Run the project using localhost or your hosting domain.

Example local URL:

```text
http://localhost/project-folder-name/
```
